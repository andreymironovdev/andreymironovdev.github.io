Tree structure and concurrent inserts

Introduction

Suppose we have a table in our web application which expresses a tree structure, for example folders and files:

create type tree_item_type as enum (''folder'',''file''); create table tree_item (id bigserial not null primary key, name text not null, item_type tree_item_type not null, parent_id bigint references tree_item(id));
